## Проект: Автономный полёт БПЛА по заданным координатам 
Цель: Реализовать алгоритм автономного полёта БПЛА по заданным координатам при помощи управления на основе ПД-регулятора.  
Технологии: Python, MuJoCo.  
### Запуск:  
1. Установить зависимости:  
   pip install -r requirements.txt
2. Запустить симулятор
    python src/main.py
